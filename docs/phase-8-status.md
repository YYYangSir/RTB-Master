# 阶段 8：完整闭环 Smoke 验证

## 目标

提供一个可重复执行的 smoke 命令，验证当前最小可演示闭环。

## 执行命令

```bash
pnpm smoke
```

该命令会：

1. 构建 API、主播管理端和用户 H5。
2. 临时启动本地 API，并等待健康检查通过。
3. 创建商品、直播间、两个用户和竞拍。
4. 启动竞拍，校验两个 Socket.IO 客户端加入和实时广播。
5. 执行多轮出价，封顶成交并校验唯一订单。
6. 查询订单并完成模拟支付，确认状态为 `PAID`。
7. 自动关闭临时 API 进程。
